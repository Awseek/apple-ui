# Apple HIG Design Patterns

## Charting Data
- Use charts to visualize data (weather, stocks, health)
- Support Bar/Line/Point marker types
- Fixed or dynamic axis ranges
- Include tick marks, grid lines, titles, annotations, legends
- Interactive: drag to view values, keyboard/switch navigation
- Full accessibility: VoiceOver + Audio Graphs

## Collaboration and Sharing
- Support real-time collaboration
- Show participant presence and changes
- Provide sharing options (link, invite, permissions)
- Handle conflict resolution gracefully

## Drag and Drop
- Support drag between apps and within app
- Provide visual feedback during drag
- Support multiple item types
- Respect content restrictions

## Entering Data
- Use appropriate input controls for data type
- Provide clear labels and placeholder text
- Validate input and show errors inline
- Support Dynamic Type for all text

## Feedback
- Provide immediate feedback for user actions
- Use haptics, sounds, and visual indicators
- Show progress for long operations
- Confirm destructive actions

## File Management
- Support standard file operations (open, save, export)
- Use system file browser when possible
- Provide clear file organization (folders, tags)
- Handle file conflicts and errors

## Going Full Screen
- Hide chrome for immersive content
- Provide easy exit gesture/button
- Maintain content aspect ratio
- Support both orientations

## Launching
- Launch quickly, show content immediately
- Use launch screen/storyboard for instant feedback
- Restore previous state when appropriate
- Handle first launch vs subsequent launches

## Live-viewing Apps
- Stream live content with minimal delay
- Provide playback controls
- Handle network interruptions gracefully
- Support picture-in-picture

## Loading
- Show loading indicators for long operations
- Use skeleton screens for content placeholders
- Provide cancel option where appropriate
- Load content progressively

## Managing Accounts
- Support Sign in with Apple
- Provide clear account creation flow
- Handle authentication errors gracefully
- Support account deletion

## Managing Notifications
- Request permission with clear explanation
- Group related notifications
- Support notification actions
- Provide notification settings

## Modality
- Use modality for focused tasks
- Provide clear way to dismiss
- Keep modal content minimal
- Support both full-screen and partial modals

## Multitasking
- Support Split View and Slide Over (iPad)
- Handle state preservation and restoration
- Respond to size class changes
- Support multiple windows

## Offering Help
- Provide contextual help when needed
- Use tooltips, coaches marks, and tutorials
- Make help discoverable but not intrusive
- Support help search

## Onboarding
- Show value proposition quickly
- Keep onboarding brief (3-5 screens)
- Allow skip option
- Request permissions in context

## Playing Audio
- Support background audio
- Provide lock screen controls
- Handle audio interruptions (calls, alerts)
- Support AirPlay and Bluetooth

## Playing Haptics
- Use haptics for tactile feedback
- Match haptic to action importance
- Support haptic customization
- Respect system haptic settings

## Playing Video
- Support full-screen playback
- Provide standard playback controls
- Support picture-in-picture
- Handle network quality changes

## Printing
- Use system print dialog
- Provide print preview
- Support page range selection
- Handle different paper sizes

## Ratings and Reviews
- Request ratings at appropriate moments
- Don't interrupt user flow
- Provide in-app feedback option
- Respect user's choice to not rate

## Searching
- Provide search suggestions
- Support recent searches
- Show results progressively
- Support filtering and sorting

## Settings
- Use system Settings app for app preferences
- Group related settings logically
- Provide clear labels and descriptions
- Support search in settings

## Undo and Redo
- Support shake to undo (iOS)
- Provide undo/redo in edit menu
- Support multi-level undo
- Show clear action descriptions

## Workouts
- Use HealthKit for workout data
- Provide real-time workout metrics
- Support workout pausing and resuming
- Handle background workout tracking
